## Iniciar el proyecto

Para levantar este microfrontend Vue, ejecuta:

```
npm run serve
```

**Importante:** Este proyecto debe ejecutarse **antes** que el proyecto root, ya que el proyecto root depende de este microfrontend para funcionar correctamente.
